# Business Orbit Scout

Python-driven pipeline to discover businesses via the Google API, loop through them in a circular collection flow, scrape websites, detect careers pages, and send extracted data to DeepSeek for analysis.

## Getting started
- Requirements: Python 3.11+
- Add environment variables for API keys before running (to be defined).
